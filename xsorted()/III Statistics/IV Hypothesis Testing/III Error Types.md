Type I & II
	Type I: reject a true null hypothesis
		probability is: $\alpha$
	Type II: accept a false null hypothesis
		probability is: $1- \beta$
				"power of the test"

|        | $H_0$ is true | $H_0$ is false |
| ------ | ------------- | -------------- |
| Accept | ok            | Type II        |
| Reject | Type I        | ok             |
